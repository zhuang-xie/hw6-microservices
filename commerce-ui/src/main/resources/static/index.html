Commerce UI

Code Group #12

Go to /items to list all items with their prices

Go to /category/{category} (e.g. /category/shoes or /category/shirts) to list all items and their prices based on a category input

Go to /item/{itemid} (e.g. /item/1001) to list an item with its price based on item id

Notes:

  • Item service runs on port 8080. It is protected by a Hystrix circuit breaker. If the circuit is tripped to open, a message "Item service not available" will be displayed.
  • Price service runs on port 8090. It is protected by a Hystrix circuit breaker. If the circuit is tripped to open, a message "Price service not available" will be displayed.
  • Commerce UI runs on port 8081.
